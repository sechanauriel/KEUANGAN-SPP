GET - Lihat Satu Mahasiswa\n\n```http\nGET /api/student/<student_id>\n```\n\n**Parameter:**\n- `student_id` (integer, required) - ID mahasiswa\n\n**Response Sukses (200):**\n```json\n{\n  \"id\": 1,\n  \"nim\": \"20241001\",\n  \"name\": \"Adi Pratama\",\n  \"email\": \"adi@email.com\",\n  \"phone\": \"081234567890\",\n  \"program_studi\": \"Teknik Informatika\",\n  \"program_studi_id\": 1,\n  \"status\": \"active\",\n  \"registration_date\": \"2024-01-15T10:30:00\"\n}\n```\n\n**Contoh CURL:**\n```powershell\ncurl http://localhost:5000/api/student/1\n```\n\n---\n\n### 3. POST - Tambah Mahasiswa Baru\n\n```http\nPOST /api/student/\nContent-Type: application/json\n```\n\n**Request Body:**\n```json\n{\n  \"nim\": \"20241234\",\n  \"name\": \"Budi Santoso\",\n  \"email\": \"budi@email.com\",\n  \"phone\": \"081234567891\",\n  \"program_studi_id\": 1,\n  \"status\": \"active\"\n}\n```\n\n**Required Fields:**\n- `nim` (string) - Nomor Induk Mahasiswa (harus unik)\n- `name` (string) - Nama lengkap mahasiswa\n- `email` (string) - Email (harus unik)\n- `program_studi_id` (integer) - ID program studi\n\n**Optional Fields:**\n- `phone` (string) - Nomor telepon\n- `status` (string) - Status: \"active\" (default), \"inactive\", \"graduated\"\n\n**Response Sukses (201):**\n```json\n{\n  \"message\": \"Student berhasil ditambahkan\",\n  \"student\": {\n    \"id\": 2,\n    \"nim\": \"20241234\",\n    \"name\": \"Budi Santoso\",\n    \"email\": \"budi@email.com\",\n    \"phone\": \"081234567891\",\n    \"program_studi\": \"Teknik Informatika\",\n    \"status\": \"active\"\n  }\n}\n```\n\n**Response Error:**\n```json\n{\n  \"error\": \"NIM 20241234 sudah terdaftar!\"\n}\n```\n\n**Contoh CURL:**\n```powershell\n$body = @{\n    nim = \"20241234\"\n    name = \"Budi Santoso\"\n    email = \"budi@email.com\"\n    phone = \"081234567891\"\n    program_studi_id = 1\n    status = \"active\"\n} | ConvertTo-Json\n\ncurl -Method POST `\n  -Uri \"http://localhost:5000/api/student/\" `\n  -Body $body `\n  -ContentType \"application/json\"\n```\n\n---\n\n### 4. PUT - Edit Data Mahasiswa\n\n```http\nPUT /api/student/<student_id>\nContent-Type: application/json\n```\n\n**Parameter:**\n- `student_id` (integer, required) - ID mahasiswa\n\n**Request Body (semua optional):**\n```json\n{\n  \"name\": \"Budi Santoso Baru\",\n  \"email\": \"budi.baru@email.com\",\n  \"phone\": \"089876543210\",\n  \"program_studi_id\": 2,\n  \"status\": \"inactive\"\n}\n```\n\n**Response Sukses (200):**\n```json\n{\n  \"message\": \"Student berhasil diupdate\",\n  \"student\": {\n    \"id\": 1,\n    \"nim\": \"20241001\",\n    \"name\": \"Budi Santoso Baru\",\n    \"email\": \"budi.baru@email.com\",\n    \"phone\": \"089876543210\",\n    \"program_studi\": \"Ekonomi\",\n    \"status\": \"inactive\"\n  }\n}\n```\n\n**Contoh CURL:**\n```powershell\n$body = @{\n    name = \"Budi Santoso Baru\"\n    phone = \"089876543210\"\n    status = \"inactive\"\n} | ConvertTo-Json\n\ncurl -Method PUT `\n  -Uri \"http://localhost:5000/api/student/1\" `\n  -Body $body `\n  -ContentType \"application/json\"\n```\n\n---\n\n### 5.
